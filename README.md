# Угадай число (Guess the Number)

## Описание

Игра "Угадай число" — мини-проект на Java Swing.  
Программа загадывает случайное число от 1 до 100, а игрок пытается его угадать.  
После каждой попытки программа подсказывает: загаданное число **больше** или **меньше** введённого.  
При угадывании выводится поздравление и количество попыток.

## Используемые технологии

- Java SE
- Swing (JFrame, JLabel, JTextField, JButton, GridLayout)
- java.util.Random

## Как запустить

1. Убедитесь, что установлен JDK 8+.
2. Скомпилируйте и запустите:

```bash
cd src
javac guess/GuessNumberGame.java
java guess.GuessNumberGame
```

## Функционал

- Генерация случайного числа от 1 до 100
- Подсказки: «больше» / «меньше»
- Счётчик попыток
- Валидация ввода (проверка на число, диапазон)
- Кнопка «Новая игра» для перезапуска
- Поддержка ввода через Enter
